# Третья ссылка LUMÉ в портфолио

Файл `index.html` — это текущая версия из репозитория `ivan-kashuba-rd.github.io` с точечным обновлением только карточки **LUMÉ Antalya — Beauty Studio**:

- описание теперь показывает три визуальных направления;
- добавлена третья кнопка **LUMÉ Riviera · Signature**;
- кнопка ведёт на `/lume-riviera/?lang=en` или `/lume-riviera/?lang=ru` в зависимости от языка портфолио.

Замените корневой `index.html` в локальной копии репозитория портфолио этим файлом, затем выполните обычные `git add`, `commit` и `push`.

Лучше сначала опубликовать репозиторий `lume-riviera`, а затем обновить портфолио: тогда новая кнопка сразу откроет работающий сайт.
